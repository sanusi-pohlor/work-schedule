# Work Schedule Board

A Kanban-style Work Schedule board application built with Vue 3, allowing users to manage tasks across different stages.

## Features
- **Kanban Board:** Drag and drop tasks between "Backlog", "Received", "In Progress", and "Completed" columns.
- **Task Management:** Add, edit, and delete tasks.
- **Task Details:** View detailed information for each task.
- **Search & Filtering:** Search for tasks by keywords or filter by their priority status (Normal, Medium, Urgent).

## Tech Stack
- **Framework:** Vue 3
- **Build Tool:** Vite
- **State Management:** Pinia
- **Styling:** Tailwind CSS & Flowbite
- **Drag & Drop:** vuedraggable

## Recommended IDE Setup

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur).
